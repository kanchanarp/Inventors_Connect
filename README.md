<br>Functionalities provided include:
<ul>
<li>Messaging</li> 
<li>Posting</li>
<li>File sharing and etc.</li>
</ul>
<br>Technologies used:
<ul>
<li>PHP</li> 
<li>HTML</li>
<li>AJAX</li>
<li>JQuery</li>
<li>CSS</li>
</ul>
